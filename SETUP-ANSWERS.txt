This file contains answers to all questions asked during the setup process.
Use this as a reference when running ./setup.sh or npm run setup-bot

ADMIN-ONLY TOOLS (Optional - Set in .env)
------------------------------------------
Restrict certain tools or categories to admins:

# Restrict specific tools
ADMIN_ONLY_TOOLS=create_github_issue,merge_pull_request,update_attio_person

# Restrict entire categories
ADMIN_ONLY_CATEGORIES=osint,crypto

# Restrict additional-tools categories to admins
# If you want "osint" tools available but admin-only:
ADDITIONAL_TOOLS=osint
ADMIN_ONLY_CATEGORIES=osint

# Or restrict specific tools from additional-tools:
ADMIN_ONLY_TOOLS=telegram_search,blockchain_lookup

# Configure different LLM models per tier
ADMIN_LLM_MODEL=claude-opus-4-5-20251101
USER_LLM_MODEL=claude-sonnet-4-5-20250929

# Restrict LLM providers per tier
ADMIN_LLM_PROVIDERS=bedrock,anthropic,vertex
USER_LLM_PROVIDERS=ollama,vertex

RECOMMENDED MINIMAL SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

For a basic working setup, you need:

1. LLM Provider (at least one):
   - Ollama (free, local) ✓ RECOMMENDED
   - Anthropic Claude (paid, best quality)
   - Google AI/Gemini (free tier available)

2. Infrastructure (automated):
   - Docker ✓ REQUIRED
   - NATS ✓ REQUIRED
   - PostgreSQL ✓ REQUIRED

3. Integration (optional but useful):
   - GitHub token (for code-related tasks)
   - Mattermost (for chat bot interface)

4. Minimal .env example:
   NATS_URL=nats://localhost:4222
   DATABASE_URL=postgresql://agent:change-this-password@localhost:5432/nimbleco
   OLLAMA_URL=http://localhost:11434
   LLM_MODEL_QUICK=qwen3.5:9b
   LLM_MODEL_CODE=qwen2.5-coder:32b
   LLM_DAILY_COST_LIMIT=10.00
   GITHUB_TOKEN=[YOUR_TOKEN_HERE]

POST-SETUP VERIFICATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

After setup completes, verify everything is running:

1. Check PM2 processes:
   pm2 list

2. View logs:
   pm2 logs

3. Check Docker containers:
   docker ps

4. Access monitoring:
   - Dashboard: http://localhost:3001
   - NATS: http://localhost:8222
   - Mattermost (if local): http://localhost:8065

5. Test the system:
   npm run task:create -- --type pr-review --pr <github-pr-url>

TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Port conflicts:
  - NATS: 4222, 8222
  - PostgreSQL: 5432
  - Mattermost: 8065
  - Dashboard: 3001

If setup fails:
  1. Check Docker is running: docker info
  2. Check logs: docker logs nimble-nats
  3. Check logs: docker logs nimble-postgres
  4. Re-run with fresh start: ./setup.sh

Reset everything:
  docker-compose down -v
  pm2 delete all
  rm -rf node_modules package-lock.json
  ./setup.sh
